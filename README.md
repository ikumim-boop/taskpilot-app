# TaskPilot — 完遂まで進めるプロジェクト管理

「完遂条件（どうなったら終わり？）」を軸にプロジェクトとタスクを管理する、インストール不要のWebアプリ（PWA）。
PMリグルド（AI管理者）の運用思想（goal / done_criteria / 次の一手 / 待ちの可視化）を、人間・チーム用にアプリ化したもの。

## 特徴
- **ホーム**: プロジェクトごとに 🟢🟡🔴 の健康状態・進捗バー・「次の一手」・⏸待ち（相手と経過日数）を表示
- **ボード**: 未着手/進行中/待ち/完了 のカンバン。ドラッグ&ドロップ（PC）・移動ボタン（スマホ）両対応
- **完遂駆動**: 全タスク完了で「🎉 完遂にする」を提案。完遂条件を満たしてクローズ
- **スマホ対応PWA**: ブラウザの共有メニュー→「ホーム画面に追加」でアプリとして起動・オフライン動作
- **データはブラウザ内のみ**（localStorage）。サーバー不要・アカウント不要。移行/バックアップはJSON書き出し/読み込み

## 使い方（配布）
静的ファイル4点（index.html / manifest.webmanifest / sw.js / icon*）をそのまま任意の静的ホスティングに置くだけ。
- ローカル確認: `python3 -m http.server 4620` → http://localhost:4620
- 公開: Netlify / GitHub Pages / Cloudflare Pages 等にフォルダごとドロップ

## 技術
- 依存ゼロの単一HTML（Vanilla JS）・ダークモード自動対応・日本語UI
- Service Worker は network-first（更新優先・オフライン時キャッシュ）

## 制限（v1）
- 端末間同期なし（JSONの手動移行のみ）。チームのリアルタイム共有はv2でバックエンド（例: Supabase）追加を検討
- 通知・リマインダーなし
